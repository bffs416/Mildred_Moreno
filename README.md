# Elite Aesthetic Clinic - Dra. Mildred Moreno

Este repositorio contiene el sitio web para la Dra. Mildred Moreno, especialista en Medicina Estética Avanzada en Bogotá, Colombia.

## Estructura del Proyecto

- `index.html`: Página principal con toda la información, servicios y contacto.
- `assets/images/`: Carpeta para imágenes del sitio (incluye imágenes de antes/después).

## Tecnologías

- HTML5
- Tailwind CSS (vía CDN)
- Google Fonts (Newsreader, Material Symbols)

## Despliegue

Este sitio está listo para ser publicado en GitHub Pages o cualquier servidor estático.
